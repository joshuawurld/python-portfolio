# Python Portfolio

A collection of Python projects and tools focused on reducing errors and turning ad-hoc checks into repeatable, reliable processes based on strong fundamentals.

## Philosophy

We build useful underlying tools that make troubleshooting frontend issues easier and reduce the ad-hoc nature of manual checks. The goal is to turn these checks into repeatable processes based on good fundamentals.

## Domains

While these tools are built to be cross-domain, the primary focus areas are those where getting it right is critical:
- **Web-based Data & Scraping**
- **Health & Public Health**
- **Healthcare Supply Chain & Resourcing Management**
